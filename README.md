# Гадание по знаку зодиака

Веб-сайт для получения ежедневных предсказаний по знаку зодиака с мистическим дизайном и адаптивным интерфейсом.

## Функционал

- Выбор знака зодиака из выпадающего списка
- Получение персонализированного предсказания на день
- Случайный выбор знака зодиака
- Просмотр характеристик знака (стихия, камни-талисманы, даты)
- Индикатор энергии знака
- Таймер до следующего гадания
- Возможность поделиться результатом в социальных сетях
- Подписка на ежедневные предсказания

## Технологии

- Чистый HTML5, CSS3, JavaScript (ES6+)
- Адаптивный дизайн (Mobile-first)
- CSS Grid и Flexbox для макетов
- CSS анимации и градиенты
- LocalStorage для сохранения последнего выбора
- Font Awesome иконки
- Google Fonts

## Установка и запуск

### Локальный запуск

1. Клонируйте репозиторий или скачайте файлы
2. Откройте файл `index.html` в браузере

### Деплой на Render

1. Создайте новый репозиторий на GitHub и загрузите все файлы
2. Зарегистрируйтесь на [Render](https://render.com)
3. Нажмите "New +" и выберите "Static Site"
4. Подключите ваш GitHub репозиторий
5. Укажите настройки:
   - **Name**: zodiac-fortune (или любое другое имя)
   - **Branch**: main
   - **Build Command**: (оставьте пустым)
   - **Publish Directory**: (оставьте пустым)
6. Нажмите "Create Static Site"
7. После сборки ваш сайт будет доступен по URL от Render
